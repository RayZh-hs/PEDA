# Physics Experiment Data Analyzer

Data analysis made easy.

---

## Introduction

PEDA (Physics Experiment Data Analyzer), is a python package that performs high-level analysis on experiment data.
Its main purpose is to automate the tedious and error-prone process of data analysis following a physics experiment.

The package is inspired by [yexi](https://github.com/stargazerZJ/yexi), a project by stargazerZJ.

The project is in development phase. Advice is welcome from all parties.
